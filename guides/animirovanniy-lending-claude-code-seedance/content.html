Собрать анимированный лендинг из Figma с Claude Code и Seedance 2.0

💡 О чём этот гайд

Собрать рабочий анимированный лендинг за 10 минут с помощью Claude Code, Higgs Field и Seedance 2.0 — от дизайна в Figma до деплоя на Cloudflare.

✅ Что ты освоишь

  1. Передавать дизайн из Figma в Claude Code (MCP или скриншот + промпт)
  2. Переходить из desktop-приложения в терминал для точечных правок
  3. Генерировать персонажей и видео в Higgs Field через Nano Banana и Seedance 2.0
  4. Убирать белый фон у AI-видео без видеоредактора
  5. Сжимать медиа одной командой Claude и пушить сайт на Cloudflare

1. Передай дизайн в Claude

Открой Figma и выбери макет. Есть два пути:

  • С подпиской Figma. Переключись в Dev Mode → найди блок MCP → скопируй example prompt. Claude получит структуру компонентов напрямую.
  • Без подписки. Сделай скриншот макета, перетащи в Claude Desktop, добавь промпт: 
    Make this into a full-screen hero section of website.
The font is Manrope. The H1 is 80 pixel size.

Выбери модель Opus 4.6 и запусти. Claude соберёт базовый HTML и CSS. Даже там где в макете нет готовых иконок — сгенерирует похожие.

2. Перейди в терминал

Desktop-версия Claude хороша для стартовой генерации. Для точечных правок попроси её:

I want to continue this with Claude in the terminal. How do I do that?

Claude выдаст команды для запуска Claude Code CLI. После этого сайт на одной половине экрана, Claude Code — на другой. Дальше всё через короткие естественно-языковые правки:

Примеры микро-правок

  • «The H1 should be regular and semi-bold weight»
  • «The star icon at the end of the 3D magic is not shown correctly»

3. Подключи SVG-ассеты

Экспортируй фон-сетку из Figma как grid.svg в папку images/. В Claude Code:

Add the grid.svg that's in the images folder to the full-screen background.
It should sit behind everything, filling the entire page.

Сетка растянется до краёв. Чтобы края не конкурировали с навигацией, добавь виньетку:

Add a radial mask to this to create a vignette mask. Tighten it up a bit.

Аналогично подключи logo.svg, button.svg, star.svg одной командой: «Replace our logo with logo.svg in images, the button icon with button.svg, the star icon with star.svg».

4. Сгенерируй персонажей в Higgs Field

Открой Higgs Field → Nano Banana 2. Промпт:

Create a Pixar style character on a white background.
He should be similar style to character from "Up".
He has a miner's helmet on and a knitted jumper.

Запускай параллельно на Nano Banana 2 и Nano Banana Pro — получишь разные вариации. Через несколько итераций подбери двух устойчивых персонажей.

Встроить в макет

Экспортируй PNG в папку images/. В Claude Code:

I want to add the two character.png images into the design.
Position them like this.

Прикрепи скриншот с финальным расположением.

5. Анимируй через Seedance 2.0

В Higgs Field переключись в Video → Seedance 2.0. Загрузи PNG персонажа и дай промпт:

This character is on a solid white background. He's looking around.
Static camera positioning. No panning.

Seedance 2.0 держит камеру статичной и не обрезает персонажа по краям. Это критично для hero-секции: движущаяся камера ломает композицию. Скачай MP4 для обоих персонажей, положи в images/ и скажи Claude:

Let's replace the images with their appropriate video in the images folder.

6. Убери белый фон без видеоредактора

Если фон сгенерированного видео не идеально белый, персонаж «поплывёт» поверх grid-сетки. Два способа:

Быстрый — CSS blend mode

Make the blend mode `darken` on the videos.

Darken выбивает белые пиксели, визуально делая видео прозрачным. Работает для 90% кейсов.

Аккуратный — DaVinci Magic Mask

  1. Перетащи видео в DaVinci Resolve
  2. Включи Magic Mask и грубо обведи персонажа
  3. Run — получишь изолированный альфа-канал
  4. Подложи белый фон и экспортируй

7. Сожми видео прямо из Claude

Два MP4 в hero-секции делают страницу тяжёлой. В терминале:

Can we compress the videos?

Claude вызовет ffmpeg с правильными параметрами. В видео character1.mp4 падает с 6 МБ до 650 КБ. Потеря качества минимальна.

8. Добавь CSS-анимации

Одним промптом:

Let's have the grid come on the screen first, grow in,
the H1 appearing one word at a time.

Claude добавит @keyframes и выстроит animation-delay. На обновлении страницы — сетка вырастает, заголовок печатается по словам.

9. Сделай адаптив

Let's make it responsive. Nav links should turn into a burger icon,
show full-screen nav pop-up with a cross at the top.
Hide the characters on mobile. Have the login button full width.

Claude сам разведёт медиа-запросы, напишет JS для бургера, скроет тяжёлые видео на узких экранах и растянет CTA по ширине.

10. Задеплой на Cloudflare

Последний промпт:

Let's push this to a GitHub repo, then show me the steps of
publishing this live on Cloudflare.

Claude создаст репозиторий через gh, запушит код и продиктует шаги для Cloudflare Pages. Подключение домена — в настройках Cloudflare за 5 минут. Хостинг для большинства случаев — бесплатный.

🔑 Ключевые мысли

  1. Desktop-Claude — быстрый старт генерации, Claude Code в терминале — точечные правки. Переход занимает один промпт.
  2. Seedance 2.0 — единственный видео-генератор, которому можно приказать «static camera, no panning» и получить именно это. Для hero-секции с персонажами критично.
  3. Blend mode darken убивает белый фон у AI-видео без видеоредактора. Быстрее DaVinci в 90% кейсов.
  4. Claude Code умеет сам сжимать медиа. Ffmpeg вручную не нужен — уменьшение в 10× без заметной потери качества.
  5. Полный путь «дизайн → задеплоенный сайт» занимает 10 минут активной работы плюс время на рендер AI-видео.
